# h5-hthty-guide

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面通常用于演示、测试、信息展示或作为轻量级前端资源。仓库本身不针对任何特定域名或网站，而是作为一个通用的静态页面集合存在。

## 目录结构

```
h5-hthty-guide/
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 公共资源（CSS、JS、图片等）
├── docs/           # 文档或说明文件（可选）
└── README.md       # 本文件
```

- **pages/**：每个独立 HTML 页面放置于此目录下，文件名通常反映其功能或用途。
- **assets/**：存放多个页面共用的样式、脚本或媒体文件，避免重复引用。
- **docs/**：可存放与页面使用相关的附加说明或参考文档。

## 页面归档说明

每个 HTML 页面均为独立完成，不依赖外部服务或数据库。页面内容可能包含静态文本、交互元素、数据可视化或简单功能演示。归档时遵循以下原则：

- 页面之间互不依赖，可单独打开运行。
- 资源文件尽量使用相对路径，确保本地或静态托管环境正常工作。
- 页面代码保持简洁，便于阅读和修改。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 使用浏览器直接打开 `pages/` 目录下的任意 HTML 文件。
3. 也可将整个仓库部署到任意静态服务器（如 GitHub Pages、Netlify 等）进行在线访问。
